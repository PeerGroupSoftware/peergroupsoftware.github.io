What it was

Peer Group Software was a startup project focused on adventurous app and game development. It started as a game development company, but due to its size, the focus shifted to making collaboration tools, a social media, a task management app, and a rideshare competitor. The only released project is YouTube-to-Mac.

Core features

  • Group spaces for socializing and finding likeminded individuals
  • Efficient programs so anyone can use the product
  • Meaningful and innovative projects and games

Key takeaways

  • Execution > ideas (every time)
  • Adoption is harder than building features
  • Systems scale — chaos doesn’t

Status

This project is intentionally frozen in time. That’s not failure — it’s a decision to keep the artifact, keep the lessons, and move forward.
